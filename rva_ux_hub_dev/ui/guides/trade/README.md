# Guia Trade - estrutura inicial

Esta pasta representa a estrutura inicial da guia `trade` do `rva_ux_hub`.

## Papel nesta fase

- existir visualmente como guia dedicada
- preparar o acoplamento futuro com o `rva_trade_engine`
- nao exigir ainda cards, paineis ou populacao operacional completa

## Regra

A guia `trade` deve ser tratada como guia especializada do modulo
`rva_trade_engine`, publicada pelo `rva_ux_hub`.
